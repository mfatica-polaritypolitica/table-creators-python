These codes produce the structure of the tables for each of the six components that make up the Enforcement Risk Score (Regulatory, Political, Legislative, Judicial, Media/NGOs, ICO Complaint Volume). These tables make up the core of the ERS model that provides the backbone for the product. The framework markdown file provides more information about all of the tables.

This set of creates the tables in the SQL database. The tables themselves can be more easily modified directly in DBeaver (or whatever SQL database management tool you choose).
